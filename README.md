# tipbox

---

支付宝通用信息提示框样式，共有提示、成功、警告、等待等七种提示框。

---

## 使用说明

### .ui-tipbox-message `提示`

### .ui-tipbox-warning `警告`

### .ui-tipbox-success `成功`

### .ui-tipbox-question `疑问`

### .ui-tipbox-stop `阻止`

### .ui-tipbox-wait `等待`

### .ui-tipbox-error `出错`

## 其他

### .ui-tipbox-white 

这是边框背景全白的样式，可与上面的样式配合使用。
